HTML Syntax


are written in Hyper Text Markup Language (HTML). HTML has three special characters: [, &, ]. An HTML command is enclosed in [... ], for example [p ], which is a paragraph separator, or [b ] ("begin bold") and [/b ] ("end bold"). So the following HTML text:
This sentence contains [b ]bold [/b ] text.
produces:
This sentence contains bold text.
